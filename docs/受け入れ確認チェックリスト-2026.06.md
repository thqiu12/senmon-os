# 受け入れ確認チェックリスト（Compass / 2026.06）

先生方が**本番画面を実際に操作して**動作を確認するためのリストです。
今回の更新（出願者タイプの追加・日本人出願の不具合修正・同時出願の上限緩和・英語表示・AI補助）と、
「学生がちゃんと出願できる」基本動線を重点的に確認します。

## 使い方
- 各項目を上から操作し、「期待どおり」なら ☑、違ったら ☐ のまま **不具合メモ欄**に書いてください。
- 不具合は **出願番号・スクリーンショット・操作した手順** を添えて報告してください（再現できると直りが早いです）。
- 分担OK：A=出願体験、B=管理画面、C=役割別、D=AI補助、で分けて進められます。
- ⚠️印は「設定（鍵）が入っていないと動かない」項目です。動かない＝故障とは限りません（担当に確認）。

## A. 学生の出願体験（最重要：出願できるか）

### A-1. 留学生で出願できる
1. 出願ページを開く → 出願者タイプで **「留学生」** を選ぶ
2. 志望校を選ぶ → フォームが表示される
3. **在留資格・在留期限・日本語レベルの欄が表示される**こと
4. 必須項目を入力 → 書類をアップロード → 確認 → 送信
5. **出願番号が画面に表示**される
6. ⚠️ 登録メールに**出願番号の確認メールが届く**（鍵未設定だと届きません）

- 期待：最後まで進めて出願番号が出る／メールが届く
- ☐ 不具合メモ：__________________________________________

### A-2. 日本人で出願できる（← 今回直した重点項目）
1. 出願ページ → 出願者タイプで **「日本人」** を選ぶ
2. 志望校を選ぶ → フォーム表示
3. **在留資格・在留期限・日本語レベルの欄が出ない**こと（日本人には不要）
4. 必須項目を入力 → 送信 → **最後まで完了して出願番号が出る**

- 期待：日本人でも送信エラーにならず、出願番号まで進める
- ☐ 不具合メモ：__________________________________________

### A-3. 出願状況の確認（再ログイン）
1. トップの「出願の続き・状況確認」から、**出願番号＋登録メール**で確認画面へ
2. 自分の申請内容・ステータスが表示される

- ☐ 不具合メモ：__________________________________________

### A-4. 英語表示
1. 出願ページの言語切替で **English** にする
2. フォームのラベル等が英語表示になる → 日本語に戻せる

- ☐ 不具合メモ：__________________________________________

### A-5. 書類の再提出（差し戻し対応）
1. 一度アップロードした書類を、差し戻し後に**再アップロード**
2. 古い書類が新しいものに**置き換わる**（二重に残らない）

- ☐ 不具合メモ：__________________________________________

### A-6. 同時出願（← 上限緩和の確認）
1. 学校のPC教室など、**複数人が同じ回線から同時に**出願してみる（できれば5〜10人以上）
2. 「申請の送信が多すぎます」等のエラーで**止められないこと**

- 期待：同時でも全員が出願できる
- ☐ 不具合メモ：__________________________________________
