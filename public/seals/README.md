# 学校印影（電子角印）

入学許可書・合格通知書・受験票に捺印する各校の公印画像を置く。

- `chuo.png`     … 中央ゼミナール
- `kanagawa.png` … 神奈川柔整鍼灸専門学校
- `tdb.png`      … 東京デジタルビジネス専門学校
- `default.png`  … 上記に該当しない場合の既定（任意）

仕様:
- 推奨: 正方形・背景白（または透明）・朱色の印影。白背景は描画時に mix-blend-mode:multiply で自然に透過する。
- 形式: png / webp / jpg いずれか（拡張子はこの順で探索）。
- 画像が無いキーは自動的に CSS 描画の角印にフォールバックする（lib/pdf/seal.ts）。
